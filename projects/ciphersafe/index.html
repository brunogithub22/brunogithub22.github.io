Project Overview

CipherSafe is a platform that uses an ESP32 microcontroller and a MicroSD card to store and encrypt various file types using the AES protocol. Communication between the microcontroller and the SD card occurs via the SPI protocol.

The web server on the ESP32 uses a self-signed certificate for secure WiFi communication. A critical feature is the safe mounting and unmounting process of the SD card, managed by a physical button to prevent data corruption.

Components Used

The core hardware and software components that power CipherSafe.

Image Component Description
ESP32 A dual-core microcontroller with built-in Wi-Fi, enabling parallel processing.
Micro SD Card Storage for files encrypted with the robust AES-256 symmetric protocol.
Push Button Triggers the safe unmounting of the SD card, ensuring data integrity upon system shutdown.
SPI SD Module A crucial component that facilitates SPI communication between the ESP32 and the SD card.
